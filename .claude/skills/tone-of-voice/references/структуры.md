# Структуры Сообщений

Паттерны построения сообщений для разных ситуаций в коммуникации с клиентами.

---

## Структура Первого Сообщения

Когда клиент только зашел в воронку:

```
1. Приветствие с именем
   "Добрый день, <Имя>!"

2. Контекст (откуда пришел запрос)
   "Мы получили Ваш запрос на нашем сайте по поводу инвестиций в недвижимость на Бали."

3. Представление + ценность компании
   "Меня зовут <Имя>, я инвестиционный консультант компании True Real Estate.
   Мы работаем по британским стандартам управления капиталом и применяем
   протокол проверки на 140 пунктов, чтобы Ваша инвестиция была выгодной и безопасной."

4. Открытый вопрос
   "Подскажите, пожалуйста, рассматриваете недвижимость на Бали для жизни или как инвестицию?"
```

## Структура Follow-up с Микро-Инсайтом

Вместо "мягкого напоминания" — давайте ценность сразу. Альтернатива стандартному follow-up:

```
"<Имя>, добрый день! Решил поделиться свежими данными нашего софта Estate Market
по району Чангу — за прошлый месяц реальная заполняемость в сегменте апартаментов
составила 82%. Если для вас актуален расчёт доходности на основе этих фактов,
а не рекламных буклетов — дайте знать, пришлю цифры."
```

**Преимущества:**
- Даёт немедленную ценность (конкретные цифры)
- Демонстрирует экспертизу (собственный софт)
- Мягко возвращает в диалог без давления

## Структура Follow-up (2-е Сообщение)

Если клиент не ответил на первое:

```
1. Приветствие с именем
   "Добрый день, <Имя>!"

2. Мягкое напоминание
   "Хотел(-а) бы напомнить о моем сообщении выше."

3. Предложение ценности (консультация)
   "Как Вы смотрите на то, чтобы получить общую консультацию с нашим экспертом,
   чтобы Вы могли понять, подходит ли Вам рынок Бали?"

4. Раскрытие ценности
   "Это займет около 30 минут, и Вы получите основную информацию:
   законы, налоги, статистику и особенности рынка."

5. Вопрос
   "Что скажете?"
```

## Структура Follow-up (3-е Сообщение)

Последняя попытка перед паузой:

```
1. Приветствие с именем
   "Добрый день, <Имя>!"

2. Проявление заботы
   "Как Ваши дела, все ли хорошо?"

3. Уточнение актуальности
   "Просто для нашего точного понимания хочу уточнить:
   тема инвестиций в недвижимость на Бали всё ещё актуальна для Вас
   или что-то поменялось?"
```

## Сбор Анамнеза

Последовательность вопросов для понимания потребностей клиента:

| Очередность | Вопрос | Почему важно |
|-------------|--------|--------------|
| 1 | Для жизни или инвестиции? | Определяет тип объектов |
| 2 | Вилла или апартамент? | Сужает выбор |
| 3 | Бюджет? | Фильтрует предложения |
| 4 | Сроки готовности к покупке? | Pre-sale vs готовые |
| 5 | Были ли на Бали? | Уровень знакомства с рынком |

**Важно:** Можно задать несколько вопросов в одном сообщении, но с объяснением:

```
"Чтобы подготовить для Вас максимально точную подборку, позвольте уточнить:
• Вы планируете покупку для проживания или как инвестицию?
• Какой ориентировочный бюджет Вы закладываете?
• Если появится идеальный объект — когда Вы будете готовы выйти на сделку?"
```

## Приглашение на Zoom

Структура эффективного приглашения:

```
1. Предложение формата
   "Предлагаю провести короткий Zoom-звонок с нашим ведущим экспертом."

2. Раскрытие ценности звонка
   "Во время встречи мы подробно обсудим Ваш запрос, поделимся инсайтами о рынке,
   юридическими особенностями и нюансами налогообложения, чтобы Вы понимали,
   чего ожидать на каждом этапе."

3. Указание времени (снимает напряжение)
   "Это займет буквально 20–30 минут."

4. Дополнительная ценность
   "Также мы подготовим персонализированную подборку объектов под Ваш запрос."

5. Предложение конкретных слотов (не "когда удобно")
   "Как Вы смотрите на то, чтобы назначить звонок на завтра — в 10:00 или 15:00?"
```

## Обработка "Скиньте Каталог"

Не отпускать клиента с каталогом — собрать информацию:

```
1. Согласие + объяснение индивидуального подхода
   "Да, конечно. Я могу подготовить Вам подборку самых выгодных инвестиций.
   Просто чтобы она была более релевантна под Ваш запрос..."

2. Сбор анамнеза (вопросы)
   "...скажите, пожалуйста:
   • Вам интересен объект для жизни или для инвестиции?
   • Вилла или апартамент?
   • Ваш приблизительный бюджет?"

3. После ответов — переход к Zoom
   "Отлично! Предлагаю короткий Zoom-звонок — покажу варианты с комментариями
   по плюсам и минусам каждого."
```

## Работа с Отложенным Спросом

Когда клиент "не готов сейчас":

```
1. Понимание и поддержка
   "Понимаю Вас — на этом этапе важно разобраться в деталях."

2. Предложение решения
   "Тогда мы можем заранее зарезервировать объект с возможностью
   отсроченного входа или зафиксировать цену в рамках pre-sale."

3. Конкретное действие
   "Давайте я подготовлю 2-3 таких варианта и покажу их на коротком Zoom-звонке."

4. Выбор времени
   "Когда Вам удобнее — в будни вечером или в выходные утром?"
```

## Уважительный Выход

Когда клиент отказывается от диалога:

```
1. Благодарность
   "Благодарим за Ваш ответ!"

2. Уважение к решению
   "Больше не будем беспокоить Вас сообщениями — полностью уважаем Ваше решение."

3. Дверь открыта
   "Если в будущем Вы снова будете рассматривать инвестиции на Бали,
   будем рады быть полезными."

4. Альтернативная связь
   "Чтобы оставаться на связи, можете подписаться на наш Instagram: [ссылка]"

5. Пожелание
   "Желаем Вам прекрасного дня и удачи в реализации Ваших планов!"
```

## Напоминание Перед Zoom

За день до встречи:

```
"Добрый день, <Имя>!
Напоминаю про Zoom-встречу завтра в [время] по [часовой пояс].
Мы подготовили лучшие предложения по [тип объекта], включая [что-то привлекательное].
Подключится [имя эксперта] — он работает на рынке Бали [N] лет.
Вас всё ещё устраивает это время?"
```

В день встречи:

```
"Добрый день! Вот ссылка на встречу, начинаем в [время] по [часовой пояс]:
[Zoom-ссылка]"
```
