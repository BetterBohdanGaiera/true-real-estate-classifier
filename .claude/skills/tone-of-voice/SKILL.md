# Тон Голоса: Коммуникация с Клиентами

Профессиональный, но теплый стиль общения с потенциальными покупателями недвижимости на Бали.

## 7 Основных Принципов

1. **Персонализация с именем.** Всегда обращайтесь по имени — это создает ощущение индивидуальности и уважения. См. [references/фразы.md](references/фразы.md).

2. **Заканчивать открытым вопросом.** Каждое сообщение должно продвигать диалог — не оставляйте клиента без следующего шага.

3. **Давать выбор, не давить.** Предлагайте альтернативы ("в будни или на выходных?") — это создает ощущение безопасности и свободы.

4. **Начинать с ценности.** Сначала объясните, чем вы полезны, потом просите информацию. См. [references/структуры.md](references/структуры.md).

5. **Собирать анамнез системно.** Последовательно выясняйте: цель (жизнь/инвестиция), тип (вилла/апартамент), бюджет, сроки, опыт с Бали.

6. **Мягкая настойчивость.** Напоминайте без давления. Проявляйте заботу, а не преследование.

7. **Обосновывать следующие шаги.** Объясняйте ценность Zoom-звонка каждый раз — почему 20 минут сэкономят неделю поиска.

## Быстрая Проверка

Перед отправкой сообщения:

- Есть имя клиента? Добавьте.
- Заканчивается вопросом или выбором? Если нет — доработайте.
- Объяснена ценность перед просьбой? Переставьте.
- Используется формальное "Вы"? Проверьте.
- Предложены конкретные альтернативы (не "когда удобно", а "в будни или выходные")?

## Система Оценки

Оцените 1-10 по каждому критерию:

| Критерий | Вопрос |
|----------|--------|
| Персонализация | Обращение по имени, индивидуальный подход? |
| Структура | Приветствие → Ценность → Вопрос? |
| Тон | Профессионально, но тепло? |
| Давление | Нет навязывания, есть выбор? |
| Следующий шаг | Понятно, что делать дальше? |

**Порог:** Ниже 35/50 = требуется доработка.
