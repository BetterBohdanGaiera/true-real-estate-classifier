# Глоссарий терминов: Недвижимость Бали

> Единый словарь терминов для работы с клиентами

---

## Формы владения

| Термин | Определение | Примечание |
|--------|-------------|------------|
| **Leasehold** | Долгосрочное право владения и распоряжения землей | До 30 лет с возможностью продления |
| **Freehold** | Полное право собственности на землю | Для иностранцев через PT PMA |
| **Sublease** | Субаренда — передача прав аренды третьему лицу | Распространено при покупке через девелопера |
| **Hak Milik** | Полное право собственности (индон.) | Только для граждан Индонезии |
| **Hak Pakai** | Право пользования (индон.) | Для иностранцев через компанию |
| **Hak Sewa** | Право аренды (индон.) | Базовая форма leasehold |

---

## Юридические термины

| Термин | Определение | Примечание |
|--------|-------------|------------|
| **PT PMA** | Иностранная компания в Индонезии (Perseroan Terbatas Penanaman Modal Asing) | Требуется для freehold |
| **Номини** | Подставное лицо для оформления собственности | Серая схема, высокие риски |
| **Лендлорд** | Собственник земли (обычно индонезиец) | Сдает землю в leasehold |
| **Due diligence** | Юридическая проверка объекта и застройщика | Обязательный этап перед покупкой |
| **Нотариальное оформление** | Официальная регистрация сделки | Сбор ~1% от стоимости |

---

## Финансовые термины

| Термин | Определение | Примечание |
|--------|-------------|------------|
| **ROI** | Return on Investment — возврат инвестиций | Выражается в % годовых |
| **Bali Rental Tax** | Туристический налог на аренду | 10% от оборота |
| **OTA** | Online Travel Agency (Booking.com, Airbnb) | Комиссия 15-18% |
| **УК** | Управляющая компания | Комиссия 15-23% |
| **Заполняемость (Occupancy)** | Процент занятых дней | Норма 65-85% |
| **ADR** | Average Daily Rate — средняя цена за ночь | Ключевой показатель |
| **RevPAR** | Revenue per Available Room | ADR x Заполняемость |
| **Net Profit** | Чистая прибыль после всех расходов | ~40% от выручки |
| **Estimated Price** | Рекомендуемая рыночная цена | Net Profit x 10 |

## Рыночные термины

| Термин | Определение | Примечание |
|--------|-------------|------------|
| **Промоушен период** | Период раскачки после запуска | 6-12 месяцев |
| **Сезонность** | Колебания спроса в течение года | Влияет на доходность |
| **High season** | Высокий сезон | Июль-август, декабрь-январь |
| **Low season** | Низкий сезон | Февраль-март |
| **Эксклюзив** | Эксклюзивное право продажи проекта | Лучшие условия для покупателя |
| **Ко-девелопмент** | Совместная разработка проекта с застройщиком | Формат работы True Real Estate |

## Типы объектов

| Термин | Определение | Примечание |
|--------|-------------|------------|
| **Вилла (Villa)** | Отдельно стоящий дом с участком | Можно freehold |
| **Апартамент** | Квартира в комплексе | Только leasehold |
| **Юнит (Unit)** | Единица недвижимости | Общее название |
| **Комплекс** | Группа объектов с общей инфраструктурой | Управляется одной УК |
| **Resort** | Курортный комплекс | Высокий класс обслуживания |

## Налоговые термины

| Термин | Определение | Ставка |
|--------|-------------|--------|
| **Налог нерезидента** | Налог для тех, кто <183 дней в Индонезии | 20% |
| **Налог резидента** | Прогрессивный налог для резидентов | 0-33% (~10% усредненно) |
| **Нотариальный сбор** | Сбор при оформлении сделки | 1% |
| **Налог на недвижимость** | Ежегодный налог на владение | $100-500/год |
| **НДС (PPN)** | Налог на добавленную стоимость | 11% (не всегда применяется) |
