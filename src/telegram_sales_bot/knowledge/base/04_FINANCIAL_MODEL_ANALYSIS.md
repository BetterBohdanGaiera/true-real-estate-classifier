# Анализ финансовых моделей застройщиков

> Как читать, проверять и выявлять "ловушки"

---

## Что такое финансовая модель

Финансовая модель — это **прогноз** доходности, основанный на:
- Аналитике рынка (лучший случай)
- "Ощущениях" застройщика (худший случай)

> **Важно:** Это не гарантия, а предположение. Задача менеджера — проверить реалистичность.

---

## Структура типичной финмодели

### Каскад расчета

```
ВЫРУЧКА (Revenue)
    │
    ├─ Bali Rental Tax (10%)
    │
    ├─ OTA комиссия (15-18%)
    │
    ├─ Комиссия УК (15-23%)
    │
    ├─ Обслуживание (service costs)
    │
    = ЧИСТЫЙ ДОХОД ИНВЕСТОРА
    │
    ├─ Налог на доход (10-20%)
    │
    = ИТОГОВАЯ ПРИБЫЛЬ
```

### Формула выручки

```
Выручка = Количество дней × Ставка за ночь × Заполняемость
```

## Чек-лист проверки финмодели

### 7 обязательных вопросов

- [ ] **1. Где Bali Rental Tax (10%)?**
- [ ] **2. Указана ли OTA комиссия?**
- [ ] **3. От чего считается комиссия УК?** (от оборота или от выручки после налогов)
- [ ] **4. Указан ли промоушен период?**
- [ ] **5. Какая формула расчета заполняемости?**
- [ ] **6. Есть ли разбивка по месяцам** или только среднегодовые?
- [ ] **7. Реалистичны ли прогнозы роста цены?**

## Типичные "ловушки" застройщиков

### 1. Не указан Bali Rental Tax

**Проблема:** Занижает расходы на 10%

**Пример принципа:** Застройщик показывает ROI 11.6%. После добавления Bali Rental Tax реальный ROI ≈ 9.5-10%.

### 2. Не указан промоушен период

**Проблема:** Завышает доходность первого года

**Принцип:** Реальный первый год = второй год по модели

### 3. Нереалистичный рост цены

**Проблема:** Обещают 30-50% роста — не гарантировано

**Принцип:** Рост цены зависит от рынка, а не от желания застройщика

### 4. Усредненные показатели

**Проблема:** Скрывают сезонность

**Принцип:** Средняя заполняемость 70% может означать 95% в пик и 40% в низкий сезон

## Промоушен период (раскачка)

### Что это

Период от технического запуска до полноценной работы объекта.

| Параметр | Значение |
|----------|----------|
| Длительность | **6-12 месяцев** |
| Доходность | Значительно ниже модели |
| В финмоделях | Почти никогда не указывается |

### Этапы раскачки

1. Технический запуск
2. Промо-бронирования (партнеры, блогеры)
3. Сбор статистики и отзывов
4. Официальный запуск
5. Публикация на OTA-агрегаторах
6. Выход на плановые показатели

> **Пример принципа:** Финмодель обещает 12% ROI с первого года. Реально: год 1 = ~5-8% (раскачка), год 2 = 12%.

## Как сравнивать финмодели разных застройщиков

### Пример принципа: сравнение двух подходов

**Застройщик A (прозрачный):**
- Bali Rental Tax: указан (10%)
- OTA: уже вычтена из выручки
- Комиссия УК: 20% от выручки после налога
- Детализация: по месяцам
- Промоушен период: не указан

**Застройщик B (непрозрачный):**
- Bali Rental Tax: **не указан!**
- OTA: 17% (указана)
- Комиссия УК: 23% от полной выручки
- Детализация: усредненно по году
- Промоушен период: указан (первый год)

### Как это влияет на ROI

При отсутствии Bali Rental Tax в расчетах ROI завышен на **1.5-2 пункта**.

## Формула оценки стоимости объекта

```
Estimated Price = Чистая прибыль за год × 10
```

Это стандартный мультипликатор для готовых объектов (10-летняя окупаемость).

### Пример принципа

| Показатель | Значение |
|------------|----------|
| Годовая выручка | $36,000 |
| Чистая прибыль (40%) | $14,400 |
| Estimated Price | **$144,000** |

---

## Красные флаги в финмоделях

### Немедленно насторожиться если:

| Флаг | Что не так |
|------|------------|
| ROI > 15% | Нереалистично для стабильного рынка |
| Заполняемость > 85% весь год | Не учитывает сезонность |
| Рост цены > 20%/год | Спекулятивный прогноз |
| Нет разбивки по месяцам | Скрывают детали |
| "Гарантированная доходность" | Кто гарантирует и чем? |

## Как проверять заявления застройщика

### Через Estate Market

1. Найти аналогичный объект в том же районе
2. Посмотреть реальную заполняемость и цены
3. Сравнить с заявленными в финмодели

### Пример принципа проверки

**Застройщик заявляет:**
- Заполняемость: 80%
- Цена: $150/ночь
- Район: Чангу

**Проверка в Estate Market:**
- Аналогичные объекты: 65-75% заполняемость
- Цены: $120-140/ночь

**Вывод:** Модель завышена на 15-20%

## Вопросы для застройщика

### Обязательные

1. "На основе каких данных построена модель?"
2. "Где Bali Rental Tax в расчетах?"
3. "Какая заполняемость в низкий сезон?"
4. "Что происходит в первый год (промоушен)?"
5. "Какие аналогичные объекты использовали для сравнения?"

### Дополнительные

6. "Кто управляющая компания?"
7. "Какой процент прямых бронирований?"
8. "Есть ли гарантии доходности и на чем основаны?"

---

## Рекомендации для менеджеров

1. **Всегда запрашивать детализацию по месяцам** — сезонность критична
2. **Проверять через Estate Market** — реальные данные vs модель
3. **Добавлять 6-12 месяцев раскачки** — к срокам окупаемости
4. **Спрашивать про Bali Rental Tax** — это главная "ловушка"
5. **Не доверять "гарантиям"** — уточнять механизм обеспечения
