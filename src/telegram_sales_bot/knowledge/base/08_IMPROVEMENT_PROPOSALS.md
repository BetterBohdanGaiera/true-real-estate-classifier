## 2. Таксономия рисков

### Текущая проблема

Риски упоминаются разрозненно в разных документах. Нет единой систематизации.

### Предлагаемая структура

**Категории рисков:**

| Категория | Примеры |
|-----------|---------|
| **Юридические** | Непродление leasehold, проблемы с PT PMA |
| **Строительные** | Задержки, качество, банкротство застройщика |
| **Рыночные** | Падение турпотока, избыток предложения |
| **Операционные** | Плохая УК, низкая заполняемость |
| **Валютные** | Курс рупии, инфляция |
| **Регуляторные** | Изменения законодательства |

**Для каждого риска указать:**
- Вероятность (низкая/средняя/высокая)
- Последствия
- Способы митигации
