# Tone of Voice: Bali Real Estate Communication

A skill for professional Russian-language communication with real estate prospects in Bali.

## What this is

This skill provides guidelines and templates for communicating with potential buyers of Bali real estate. It covers the full communication cycle: from first contact to Zoom invitation, including handling objections and non-responsive clients.

## Core Principles

1. **Personalize with name**  - Always address by name
2. **End with open questions**  - Keep dialogue moving
3. **Give choices, don't pressure**  - "Weekdays or weekends?"
4. **Lead with value**  - Explain why it's useful before asking
5. **Collect info systematically**  - Purpose, type, budget, timeline
6. **Soft persistence**  - Gentle reminders, not harassment
7. **Justify next steps**  - Always explain Zoom value
