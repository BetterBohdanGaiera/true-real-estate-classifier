## Bali Rental Tax (Туристический налог)

### Параметры

| Параметр | Значение |
|----------|----------|
| Ставка | **10%** от оборота |
| База | Вся выручка **ДО** вычета OTA комиссий |
| Применяется | К посуточной и долгосрочной аренде |
| Кто платит | Управляющая компания (удерживает) |

> **Критически важно:** Не все застройщики указывают этот налог в финмоделях. Всегда спрашивать: "Где Bali Rental Tax?"

### Практика рынка по оптимизации

| Способ | Кто использует |
|--------|----------------|
| Полная декларация | Крупные европейские УК (Mira, Anantara) |
| Пониженная ставка | Крупные компании с индивидуальными договорами |
| Криптовалюта | Выплаты на крипто-кошелек |
| Наличные | Без регистрации чеков |

## Каскад расчета доходности

```
ВЫРУЧКА (Revenue)
    │
    ├─ минус Bali Rental Tax (10%)
    │
    ├─ минус OTA комиссия (15-18%)
    │
    ├─ минус Комиссия УК (15-23%)
    │
    ├─ минус Расходы на обслуживание
    │
    = ЧИСТЫЙ ДОХОД ИНВЕСТОРА
    │
    ├─ минус Налог на доход (10-20%)
    │
    = ИТОГОВАЯ ПРИБЫЛЬ
```

### Формула расчета выручки

```
Выручка = Количество дней × Ставка за ночь × Заполняемость
```

**Пример:** 30 дней × $129 × 78% = $3,018/месяц

|--------|--------|
| Комиссия агентства | 5% |

| Статья | Ставка |

| Статус продавца | Ставка налога |
|-----------------|---------------|
| Нерезидент | **20%** от суммы продажи |

| Нотариальная передача | 1% |

На рынке распространена практика указания заниженной цены в договоре для минимизации налогов.

> **Риск:** На продавце и покупателе.

## Быстрые расчеты

### Грубая оценка чистой доходности

```
Чистая доходность ≈ Выручка × 0.4
```

(после всех налогов и комиссий)

### Оценка стоимости объекта

```
Estimated Price = Чистая годовая прибыль × 10
```

(стандартный мультипликатор для готовых объектов)

## Рекомендации

### Для анализа финмоделей

1. Всегда запрашивать детализацию по месяцам
2. Спрашивать про Bali Rental Tax
3. Уточнять базу для расчета комиссии УК
4. Добавлять 6-12 месяцев раскачки к срокам окупаемости

### Для получения дохода

1. Определиться со способом за 3-4 месяца до сдачи
2. Обсудить с УК опции (SWIFT/крипто)
3. Учитывать налоговое резидентство в своей стране
