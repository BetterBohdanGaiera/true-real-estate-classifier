### 4. Формулы и расчеты

#### Выручка (Revenue)
```
Выручка = Дней × Загрузка × Ставка
Пример: 31 × 0.72 × $116 ≈ $2,560
```

#### Bali Rental Tax (Налог)
```
Налог = Выручка × 10%
Ячейка: =F8*C28 (где C28 = 10%)
Пример: $2,560 × 10% = $256
```
**ВАЖНО:** Налог считается ДО вычета комиссии OTA (booking.com)!

#### Выручка* (после налога)
```
Выручка* = Выручка - Налог
Пример: $2,560 - $256 = $2,304
```

#### Комиссия управляющей компании
```
Комиссия УК = Выручка* × 20%
Ячейка: =H8*D29 (где D29 = 20%)
Пример: $2,304 × 20% = $461
```

#### Чистый доход
```
Чистый доход = Выручка* - Комиссия УК - Обслуживание
Пример: $2,304 - $461 - $512 = $1,331
```
