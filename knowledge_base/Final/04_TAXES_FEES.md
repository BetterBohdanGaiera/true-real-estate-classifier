## Bali Rental Tax (Туристический Налог)

### Параметры

| Параметр | Значение |
|----------|----------|
| Ставка | **10%** от оборота |
| База | Вся выручка **ДО** вычета OTA комиссий |
| Применяется | К посуточной и долгосрочной аренде |
| Кто платит | Управляющая компания (удерживает) |

> **КРИТИЧЕСКИ ВАЖНО:** Не все застройщики указывают этот налог в финмоделях. Всегда спрашивать: "Где Bali Rental Tax?"

### Практика Рынка по Оптимизации

| Способ | Кто использует |
|--------|----------------|
| Полная декларация | Крупные европейские УК (Mira, Anantara) |
| Пониженная ставка | Крупные компании с индивидуальными договорами |
| Криптовалюта | Выплаты на крипто-кошелек |
| Наличные | Без регистрации чеков |

## Каскад Расчета Доходности

```
ВЫРУЧКА (Revenue)
    |
    |-- минус Bali Rental Tax (10%)
    |
    |-- минус OTA комиссия (15-18%)
    |
    |-- минус Комиссия УК (15-23%)
    |
    |-- минус Расходы на обслуживание
    |
    = ЧИСТЫЙ ДОХОД ИНВЕСТОРА
    |
    |-- минус Налог на доход (10-20%)
    |
    = ИТОГОВАЯ ПРИБЫЛЬ
```

### Формула Расчета Выручки

```
Выручка = Количество дней x Ставка за ночь x Заполняемость
```

**Пример:** 30 дней x $129 x 78% = $3,018/месяц

## Примеры Расчетов (Исторические Данные)

> **ВАЖНО:** Это общие ориентиры для понимания концепта. Точные расчеты зависят от конкретного объекта.

### Пример Расчета Чистой Доходности

На примере типичного апартамента:

```
Выручка (месяц): ~$3,000
|-- Bali Rental Tax (10%): -$300
|-- OTA комиссия (~17%): -$510
|-- Комиссия УК (~20%): -$600
|-- Обслуживание: -$200
= Чистый доход: ~$1,390/месяц

Годовой чистый доход: ~$16,680
При стоимости объекта $150,000:
ROI ~ 11% годовых
```

> Это упрощенный пример для понимания механики. Реальные показатели зависят от множества факторов.
