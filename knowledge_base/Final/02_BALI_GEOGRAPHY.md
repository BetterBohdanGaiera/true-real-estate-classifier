# География Бали: Регионы для Инвестиций

> Неизменные характеристики регионов, инфраструктура, типы туристов

---

## Общая Характеристика Острова

| Параметр | Значение |
|----------|----------|
| Застроенная часть | ~5% территории |
| Основные районы | 4 (Чангу-Кута, Убуд, Букит, Санур) |
| Ограничение высотности | Максимум 15 метров |
| Рабочая сила | Только местные (иностранцы запрещены) |
| Типичный проект | 50-100 юнитов (vs 300-400 в Дубае/Таиланде) |
| Новых юнитов в год | ~3-4 тысячи на весь остров |

### Бали как Мировой Бренд

- Бали известен более 100 лет (в отличие от Дубая ~20 лет)
- Турпоток стабильный, восстановился после COVID за 3-4 года
- Власти уже ограничивают поток дополнительными сборами
- Культура возвращения — туристы приезжают повторно

## Агломерация Чангу-Кута-Семиньяк

### Структура (с юга на север)

| Район | Характеристика | Целевая аудитория |
|-------|----------------|-------------------|
| **Кута** | Старый, бюджетный, бары | Небогатые туристы, тусовщики |
| **Семиньяк** | Ночная жизнь, бары | Молодежь, тусовщики |
| **Чангу** | Рестораны, серфинг, йога | Digital nomads, молодежь |
| **Перереран** | Широкие дороги, спокойно | Семьи с детьми |
| **Сесех** | Удаленный, транспортные риски | Ограниченно рекомендуется |

### Характеристики Чангу

| Аспект | Описание |
|--------|----------|
| Главное преимущество | Лучшие рестораны острова |
| Формат отдыха | Утром йога/серфинг, днем еда, вечером 2 коктейля |
| Ночная жизнь | В 23:00 уже пусто (тусовки в Куте/Семиньяке) |
| Инфраструктура | Узкие улицы, пробки, но расширяется |
| Пляжи | Грязные (мусор из Индийского океана) |

### Преимущества Перерерана

- Широкие дороги с тротуарами
- Можно гулять пешком с детьми
- 4 минуты до центра Чангу
- Спокойная семейная атмосфера
- Активное развитие ресторанов

### Транспортная Проблема Сесеха

> **КРИТИЧЕСКИЙ РИСК:** До Перерерана 3 альтернативные дороги. После — только одна.

- Сесех = "one way trip"
- 30+ минут обратно vs 4 минуты из Перерерана
- Инфраструктура ограничена

## Полуостров Букит

### Богатые Районы

#### Нуса-Дуа

| Аспект | Описание |
|--------|----------|
| Формат | Закрытая территория со шлагбаумом |
| Инфраструктура | 20+ крупных отелей, единая УК |
| Дороги | Широкие, газоны, тротуары |
| Аудитория | Семьи, пенсионеры |
| Цены | От $150/ночь, дешевого нет |

#### Миласти-Пандава

- Самый красивый и большой пляж Бали
- Дорогие сетевые отели
- **Ниша:** нет отелей $100-150/ночь

### Бюджетные Районы

#### Улувату

| Аспект | Описание |
|--------|----------|
| Расположение | Левая часть полуострова |
| Характер | Серферские споты, обрывы, закаты |
| Цены | $50-120/ночь |
| Особенность | Мало дорогого сегмента |

#### Джимбаран

| Аспект | Описание |
|--------|----------|
| Расположение | Ближе к аэропорту |
| Аудитория | Небогатые индонезийцы |
| Цены | $40-70/ночь |
| Пляжи | Красивые, но без закатов (загорожены) |

### Центр Букита

- Малопривлекателен для инвестиций
- Исключение: виллы на горе с видом на океан
- Вид компенсирует удаленность от пляжа

## Убуд — Духовный Центр

### Общая Характеристика

| Аспект | Описание |
|--------|----------|
| Природа | Самый зеленый регион, как "Аватар" |
| Развитие | Равномерно во все стороны |
| Виды | Рисовые поля, каньоны, водопады |
| Климат | Прохладнее, чем на побережье |
| Храмы | Большинство храмов Бали здесь |

### Уникальные Особенности

- Каньоны глубиной до 100 метров
- Рестораны с видом на рисовые поля
- Невероятные закаты
- Духовная атмосфера

### Перспективные Зоны

Районы между центром Убуда и Кинтамани:
- 15-20 минут от центра
- Еще более впечатляющая природа
- Меньше застройки
- Огромные каньоны

---

## Кинтамани

### Для Туризма

- Старый вулкан с маленьким внутри
- Красивые рестораны на хребте
- 40-60 минут от Убуда

### Для Инвестиций

> **НУЛЕВЫЕ ПЕРСПЕКТИВЫ:** Туристы приезжают на полдня и уезжают. Нечего делать дольше 3 часов.

## Санур и Денпасар

### Санур

| Аспект | Описание |
|--------|----------|
| Формат | Часть Денпасара на побережье |
| Аудитория | Семьи, пенсионеры |
| Отели | Богатые, персональные пляжи |
| Серфинг | Споты далеко от берега, чистая вода |
| Ощущение | Город у моря |

### Денпасар

- Обычный город, не туристический
- Торговые центры, цивилизация
- Не представляет интереса для инвестиций

| Параметр | Значение |
|----------|----------|

- Еще дальше, чем Чандидаса
- Классный сноркелинг
- Красивые вулканы и пляжи
- Нулевые инвестиционные перспективы

## Проблема Качества Жилья

### Дисбаланс Рынка

| Сегмент | Ситуация |
|---------|----------|
| Дешевое ($200-300/мес) | Огромное количество |
| Качественное ($3000+/мес) | Острый дефицит |

### Причины

1. Плохая гидроизоляция старых объектов
2. Влажный климат — плесень за 1-2 года
3. Убыль номерного фонда выше прироста
4. Качественное строительство только последние 4-5 лет

> **ВЫВОД:** Качественный объект служит долго и приносит прибыль. Некачественный — не продать и не сдать.

## Работа с Возражениями по Географии

**"На Бали слишком много строят"**
- Проекты маленькие из-за ограничений
- Убыль старых объектов > прироста новых
- Качественного жилья острый дефицит

**"Видел дешевые варианты на Airbnb"**
- Смотрите отзывы о состоянии
- Дешево = плесень, сырость
- Турист с бюджетом не экономит на качестве
