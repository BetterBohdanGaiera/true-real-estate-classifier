## Ключевые Формулы

### Расчет выручки
```
Выручка = Дни x Цена/ночь x Заполняемость
```

### Расчет чистой доходности
```
Чистый доход = Выручка - Bali Tax (10%) - OTA (15-18%) - УК (20-23%) - Обслуживание
```

### Грубая оценка чистой доходности
```
Чистая доходность ~ Выручка x 0.4
```
(после всех налогов и комиссий)

### Оценка стоимости объекта
```
Estimated Price = Чистая годовая прибыль x 10
```
(стандартный мультипликатор для готовых объектов)

## Интерпретация Заполняемости

| Заполняемость | Значение | Действие |
|---------------|----------|----------|
| < 50% | Слабый объект или новый | Проверить причины |
| 50-70% | Норма | Стандартная ситуация |
| 70-85% | Хорошо | Оптимальный диапазон |
| **> 90%** | Цена занижена | Можно повысить цену на 15-20% |

> При заполняемости 97% можно повысить цену на 20%, снизить заполняемость до 80% и заработать БОЛЬШЕ при меньшем износе.

## Быстрые Ответы

**"Напрямую дешевле?"**
> Нет, цены одинаковые. Застройщик не может продавать дешевле агентов — иначе потеряет каналы продаж.

**"Зачем агентство?"**
> Независимый партнер + помощь с перепродажей + контроль стройки. И бесплатно — комиссию платит застройщик.

**"Leasehold — это аренда?"**
> Это полноценное владение с правом продажи, субаренды и наследования. В Лондоне 90% центра — leasehold.

**"Какие гарантии доходности?"**
> Гарантий нет — это инвестиция. Но есть данные Estate Market по реальной доходности аналогов.
