## Ключевые инсайты

### Для продаж

1. **Бали — мировой бренд уровня Нью-Йорка и Лондона**
2. **True Real Estate — не агентство, а инвестиционный партнер**
3. **0% комиссии для клиента** — застройщик платит
4. **AI-аналитика Estate Market** — уникальное преимущество
5. **Собственные инвестиции в проекты** — "мы в одной лодке"

### Для анализа рынка

1. **Только 5% острова застроено** — потенциал роста
2. **Убыль объектов > прироста новых** — дефицит качественного жилья
3. **Заполняемость >90% — плохо** — можно поднять цену
4. **Bali Rental Tax 10%** — не все застройщики указывают
5. **Промоушен период 6-12 месяцев** — первый год = второй год модели

### Для юридических вопросов

1. **Freehold через PT PMA** — требует $170K+ капитала
2. **Leasehold — стандарт рынка** для иностранцев
3. **Продление аренды** — зависит от статуса земли
4. **Налог при перепродаже** — 20% для нерезидентов
